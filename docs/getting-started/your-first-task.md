## Key Concepts

### Ask vs. Do

1M Code can either:
- **Answer questions** â€” Explain concepts, give advice, help you think
- **Take actions** â€” Write code, create files, run commands

In early modes (Idea, Spec), it focuses more on questions. In later modes (Build, Ship), it takes more actions.

### Context Matters

1M Code remembers your conversation. You can say things like:
- "Actually, let's make it for students instead"
- "What did you mean by that?"
- "Can you explain that simpler?"

It understands context and adjusts accordingly.

### Asking for Help

At any point, you can ask:
- "I'm confused, can you explain this differently?"
- "What should I do next?"
- "I'm stuck, help!"

1M Code is designed for beginners. Don't be afraid to ask "dumb" questionsâ€”there aren't any.
